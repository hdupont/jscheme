Jscheme

jscheme est le portage en Javascript (source) de Charme, un interpréteur d'un sous-ensemble de Scheme écrit en Python (source) par David Evans pour son livre Introduction to Computing Explorations in Language, Logic, and Machines.

A votre disposition: +, -, *, /, <, define, lambda, if

NB. La valeur d'un define n'est pas définie.

Pour essayer jscheme, copiez une des définitions suivantes dans le champs "Code à exécuter" ci-dessous et cliquez sur Exécuter

(define factorial
    (lambda (x)
        (if (= x 0)
            1
            (* x (factorial (- x 1))))))

(factorial 4)

ou encore en CPS (Continuation-Passing Style)

(define c-fact (lambda (n continue)
    (if (= n 1)
        (continue 1)
        (c-fact (- n 1)
            (lambda (fact-n-1)
                (continue (* n fact-n-1)))))))

(c-fact 4 (lambda (v) v))
